Question-1: What are the different ways to select an element in the DOM?

Answer: There are many ways to select an element in the DOM.Such as :By tag name(document.getElementsByTagName),By id(document.getElementById),By class name(document.getElementsByClassName) etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:
inner.Html:The HTML content of an element, represented as a string.

inner.Text: It is used to access what is written inside an element.

inner.Content: It represents all the text in an element

Question-3: What is event delegation in the DOM?

Answer: Event Delegation is a pattern that handle events efficiently.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a process where when an event is triggered on a child element, it propagates to its parent and higher elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:
Create Child: document.createElement("div");
adding: document.body.appendChild(newDiv);
remove: parent.removeChild(child);
